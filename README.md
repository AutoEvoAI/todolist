# 星空Todo Pro

一个美观实用的待办事项管理应用，带有番茄钟功能。

## 功能特点

- **任务管理** - 添加、完成、删除待办事项
- **番茄钟** - 25分钟专注计时器，帮助提升专注力
- **时间追踪** - 记录每个任务花费的番茄时间
- **数据持久化** - 使用 localStorage 保存所有数据
- **视觉效果** - 星空主题设计，美观的动画效果

## 使用方法

1. 直接在浏览器中打开 `todo-final-stable.html` 文件
2. 在输入框中输入任务，按 Enter 或点击"添加"按钮
3. 点击任务右侧的番茄按钮启动专注计时
4. 点击勾选按钮完成任务
5. 点击垃圾桶按钮删除任务

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需安装任何依赖
- 使用 localStorage 存储数据

## 界面预览

- 深色星空背景，带有动态星星效果
- 半透明毛玻璃效果的任务卡片
- 醒目的番茄钟计时器
- 任务统计展示（总番茄数、今日番茄数）
